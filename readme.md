# Lógica de Programação - Projeto Enem(Prova)

## Pré-requisitos
- Visual studio 
- Python

## Objeto 
Esse projeto tem por objeto inicial solicitar ao usuário as informações de nome, e-mail e senha para se cadastrar, após isso informar usuário e senha para logar, caso a senha seja incorreta será solicitada novamente até que a senha seja correta, após isso será solicitado o nome das 5 matérias do Enem e as notas correspondentes, por fim será apresentada a média das notas e se com a média das notas o usuário conseguiu ganhar alguma bolsa de desconto de acordo com as faixas de notas e qual o valor da mensalidade com desconto no primeiro ano.

### Faixas 
0 a 200 Pontos -> Sem desconto  
201 a 400 Pontos -> 20% de desconto  
401 a 600 Pontos -> 40% de desconto  
601 a 800 Pontos -> 60% de desconto  
801 a 900 Pontos -> 80% de desconto  
901 a 1000 Pontos -> 100% de desconto  

## Funcionalidade

A principal função desse projeto é informar a média das notas do Enem, através da inserção manual do usuario e verificar se ele conseguiu alguma bolsa de desconto.
